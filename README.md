# 공유재산 관리 플랫폼

공공 재산을 시민들이 쉽게 검색하고 예약할 수 있는 웹 플랫폼입니다.

## 📋 프로젝트 개요

이 플랫폼은 지자체가 보유한 공유재산(공공시설, 공원, 회의실, 장비 등)을 효율적으로 관리하고, 시민들이 편리하게 검색하여 예약할 수 있도록 지원합니다.

## ✨ 주요 기능

### 시민용 기능
- 🗺️ **지도 기반 검색**: 카카오맵을 활용한 직관적인 재산 검색
- 🔍 **다양한 필터**: 지역, 카테고리, 상태별 필터링
- 📅 **온라인 예약**: 간편한 예약 신청 시스템
- ⭐ **리뷰 시스템**: 사용 후기 작성 및 평점 제공
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원

### 관리자 기능
- 📊 **대시보드**: 실시간 통계 및 현황 파악
- 🏢 **재산 관리**: 재산 등록, 수정, 삭제
- ✅ **예약 승인**: 예약 신청 승인/거부
- 👥 **사용자 관리**: 회원 정보 관리
- 📈 **통계 분석**: 사용 현황 및 트렌드 분석

## 🛠️ 기술 스택

### 프론트엔드
- HTML5, CSS3, JavaScript (ES6+)
- Kakao Maps JavaScript API
- Chart.js (통계 차트)

### 백엔드
- PHP 7.4+ (또는 8.x)
- MySQL 8.0
- PDO (데이터베이스 연결)

### 서버
- Apache 2.4+ 또는 Nginx
- mod_rewrite (URL 재작성)

## 📦 설치 방법

### 1. 시스템 요구사항

```bash
- PHP 7.4 이상
- MySQL 8.0 이상
```

## 🚀 실행

## 📱 반응형 디자인

- 데스크톱: 1024px 이상
- 태블릿: 768px ~ 1023px
- 모바일: 767px 이하

## 🔒 보안 권장사항

1. **비밀번호 변경**: 기본 관리자 비밀번호를 즉시 변경하세요.
2. **SQL 인젝션 방지**: PDO prepared statements 사용
3. **XSS 방지**: `htmlspecialchars()` 함수 사용
4. **CSRF 토큰**: 폼 제출 시 토큰 검증 추가 권장
5. **HTTPS 사용**: SSL 인증서 설치 권장
6. **파일 업로드 검증**: 허용된 파일 형식만 업로드

## 🐛 문제 해결

### 지도가 표시되지 않는 경우
- 카카오맵 API 키가 올바르게 설정되었는지 확인
- 브라우저 콘솔에서 에러 메시지 확인
